# Product Service API

A RESTful Spring Boot application for managing products with full CRUD support, JWT-based authentication, and PostgreSQL as the database.

## Tech Stack

- Java 17
- Spring Boot 3
- PostgreSQL
- Gradle
- JWT Security
- Spring Data JPA
- Swagger UI
- Docker

## Features

- JWT Authentication (`/api/v1/auth/login`)
- CRUD operations for Product
- Global Exception Handling
- API Documentation (Swagger)
- Logging with `java.util.logging`

## Getting Started

### Prerequisites

- Java 17+
- Docker
- Gradle (optional, wrapper is included)
